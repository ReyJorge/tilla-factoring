# Credit Memo Template (DRAFT — NOT APPROVED FOR PRODUCTION)

Use this structure when writing the narrative portion of a recommendation.

## Sections

1. **Parties:** Supplier name / ICO — Anchor name / ICO  
2. **Facility:** Invoice amount, requested advance %, currency assumptions  
3. **Financial snapshot:** Revenue, EBITDA, P/L, receivables, payables (with dates/sources)  
4. **Exposure:** Existing supplier / anchor exposures  
5. **Behavioral:** Overdue information summary  
6. **Score:** External scoring grade **A–D** (source cited)  
7. **Risks & mitigants:** Bullet lists  
8. **Conditions precedent:** CP checklist  
9. **Recommendation:** Approve / Conditional / Reject / Human review — **non-binding**

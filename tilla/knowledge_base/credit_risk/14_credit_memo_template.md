# Credit memo template — narrative sections

AI JSON mirrors diligence outline:

1. **executive_summary** — cite gate + band + concentration verbatim.
2. **supplier_assessment** — financial disclosures completeness / gaps (`missing_information`).
3. **anchor_assessment** — rating path + confirmation/legal posture (`confirmation_status`, `legal_status`).
4. **invoice_assessment** — eligibility bullets referencing checklist triggers without inventing ERP facts.
5. **scoring_assessment** — explain drivers (`paid_invoice_count`, penalties) strictly from `model_result`.
6. **concentration_assessment** — interpret flags already emitted.
7. **key_risks / mitigants / conditions_before_funding** — mirror deterministic outputs where factual.

Never fabricate jurisprudence; legal conclusions belong to counsel offline.
